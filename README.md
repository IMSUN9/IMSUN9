##  About Me

안녕하세요.  
저는 **백엔드 개발자**를 목표로 꾸준히 공부하고 있는 **임선구**입니다.

단순히 기능을 따라 만드는 데서 끝나지 않고,  
**왜 이런 구조로 동작하는지 이해한 뒤 직접 구현하고 끝까지 완성하는 과정**을 중요하게 생각합니다.

현재는 **Java / Spring Boot / MySQL** 기반으로  
인증, 일정 관리, 사용자별 데이터 처리 흐름이 포함된 웹 서비스를 직접 만들며 성장하고 있습니다.

##  Featured Project

### JWT 인증 기반 일정 관리 캘린더 웹앱

사용자 인증과 일정 관리를 결합한 웹 프로젝트입니다.  
프론트엔드와 백엔드를 직접 연결하면서,  
**로그인 → 인증 → 사용자 식별 → 일정 저장 / 조회 / 삭제 / 완료 처리**의 흐름을 구현했습니다.

#### Main Features
- 회원가입 / 로그인
- JWT 기반 사용자 인증
- 사용자별 일정 저장 / 조회 / 삭제
- 일정 완료 여부 처리
- 일정 검색 기능
- 오늘 날짜로 이동 기능
- 일정 통계 보기
- 카카오 로그인 및 알림 기능 확장 진행 중

#### Tech
`Java` `Spring Boot` `Spring Security` `JWT` `MySQL` `HTML` `CSS` `JavaScript`
