## 🚀 Featured Project

### JWT 인증 기반 일정 관리 캘린더 웹앱

사용자 인증과 일정 관리를 결합한 웹 프로젝트입니다.  
직접 백엔드와 프론트엔드를 연결하면서  
**로그인부터 일정 저장, 조회, 삭제, 완료 처리까지의 흐름**을 구현했습니다.

#### 주요 기능
- 회원가입 / 로그인
- JWT 기반 사용자 인증
- 사용자별 일정 저장 / 조회 / 삭제
- 일정 완료 여부 처리
- 일정 검색 기능
- 오늘 날짜로 이동 기능
- 일정 통계 보기

#### 사용 기술
- Java
- Spring Boot
- Spring Security
- JWT
- MySQL
- HTML / CSS / JavaScript

#### 확장 예정
- 카카오 로그인
- 일정 알림 기능
- 서비스 구조 개선 및 프로젝트 고도화
